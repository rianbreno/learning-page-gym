# Life Fit - Academia

Este repositório contém o código-fonte do site **Life Fit**, um site institucional para uma academia, apresentando informações sobre serviços, unidades, planos e formas de contato.

## 📌 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
